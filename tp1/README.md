# TP1 – CI1062 (UFPR)

## 📌 Descrição
Este projeto implementa operações básicas com números racionais em C.  
O programa sorteia pares de racionais e realiza as operações de soma, subtração, multiplicação e divisão, tratando casos especiais como:
- Racional inválido (denominador zero no sorteio)
- Número inválido (resultado de operação com denominador zero)
- Divisão inválida (tentativa de dividir por zero)

## 📂 Estrutura de Arquivos
- `tp1.c` → programa principal (loop de execução e impressão dos resultados)
- `racional.c` → funções para manipulação de racionais (soma, subtração, multiplicação, divisão, simplificação)
- `racional.h` → cabeçalho com definições e protótipos
- `Makefile` → regras de compilação
